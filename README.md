# Instalacion

```bash
$ npm install
```

## Ejecutar en desarrollo
1. Clonar al repositorio
2. Levantar la base de datos
```
docker-compose up -d
```

3. Levantar la aplicacion en diferentes modos
```bash
# development
$ npm  start

# watch mode
$ npm start:dev

# production mode
$ npm start:prod
```
